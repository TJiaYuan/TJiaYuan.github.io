8. 作用域

8.1 作用域概述

  • 通常来说，就是一段程序代码限定可用范围，目的是为了提高程序的可靠性，减轻命名冲突。
  • 全局作用域：在整个script标签或者js文件
  • 局部作用域：在函数内部的就是局部作用域，只在函数内生效或起作用

8.2 变量的作用域

  • 根据作用域的不同分为全局变量和局部变量

8.2.1 全局变量

8.2.2 局部变量

8.2.3 全局变量和局部变量的区别

  • 全局变量只有在浏览器关闭的时候才会销毁，比较占内存资源
  • 局部变量在程序执行完毕就会销毁，比较节约内存资源

8.2.4 JavaScript没有块级作用域

  • JavaScript 也是在 ES 6 才有块级作用域
  • 块级作用域 { }

8.3 作用域链

  • 只要写代码，就一定有作用域
  • 写在函数内部的局部作用域
  • 如果函数还有函数，那么在这个作用域就可以生成多一个作用域
  • 根据内部函数可以访问外部函数变量的机制，用链式查找决定哪些数据能被内部函数访问，就称作作用域链

8.4 JavaScript预解析

  • js引擎运行js，预解析，代码执行
  • 预解析，js引擎会把js里面所有的var还有function提升到当前作用域的最前面
  • 代码执行，按照代码书写的顺序从上往下执行

9. 对象

  • 对象是一组无序的相关属性和方法的集合，所有事物都是对象，例如字符串、数值、数组、函数等。
  • 属性：事物的特征，在对象中的用属性来表示
  • 方法：事物的行为，在对象中用方法来表示

9.1 为什么需要对象

  • 保存一个值时，可以使用变量，保存多个值（一组值）时，可以使用数组。
  • 对象使表达结构更加清晰

  • 变量 单独声明并赋值，使用的时候直接写变量名 单独存在
  • 属性 在对象里面不需要声明，使用时必须 对象.属性

  • 函数 单独声明并且调用。 函数名( )

9.5 new 关键字
